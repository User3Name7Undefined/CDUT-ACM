### 题目背景

小M喜欢玩一款叫作《世界》的游戏，《世界》是一个由方块组成的三维空间。

我们可以将《世界》的大小表示为 $X,Y,Z$。内部的位置集合为 $\{(x,y,z)|{x\in[0,X-1],y\in[0,Y-1],z\in[0,Z-1]}\}$ 。

《世界》中有地面和三种方块：沙子、石头和火把。

你可以把地面理解为一种由特殊的同种方块构成的无限大平面，甚至可以延申到《世界》外部。

地面可以表示为$\{(x,y,z)|z=-1\}$ 。

《世界》有一套特殊的物理法则：

- 方块可以被放置，当且仅当该位置在《世界》内部 且为没有方块 且相邻六个面的位置已存在方块或为地面。

- 当方块的正下方不是地面或者方块，我们认为这个方块是悬浮的。

- 石头可以悬浮而不会落下。

- 沙子悬浮会下落。

- 火把是一种特殊的方块，只能 **附着** 在已经存在的方块的侧面或上面。

- 你可以将 **附着** 理解为一种特殊的放置。

- 附着关系一旦建立就不会改变。

- 火把不能附着在火把上。

- 当火把附着的方块被破坏/下落，该火把立刻消失。

显然，当没有沙子悬浮的时候，《世界》是稳定的。

### 题目描述

小M在心中构想了一份计划。计划分为 $N$ 条指令，每一条可能是放置或者破坏。

小M想让你检测计划的可行性。你将按顺序检测每条指令是否合法。

如果当前指令合法，执行之，输出一行GOODJOB，等待《世界》稳定再进行下一步;

否则，跳过当前指令，输出一行 AREYOUKIDDINGME。

所有指令检测完后，输出《世界》的状态，具体格式见下文。

指令有以下几种：

---

##### 放石头/放沙子

输入格式为一个字符串和三个整数。

$PUT\_STONE\ x\ y\ z$ 参数表示放置位置

$PUT\_SAND\ x\ y\ z$ 参数表示放置位置

如果放置位置没有方块且相邻六个面的位置存在方块或为地面，则合法。

---

##### 插火把

输入格式为一个字符串和四个整数。

$PUT\_TORCH\ x\ y\ z\ f$ 参数表示放置位置和附着的方向。

$f$ 有以下几种取值

* 1：附着在坐标为 $(x+1,y,z)$ 上

* 2：附着在坐标为 $(x-1,y,z)$ 上

* 3：附着在坐标为 $(x,y+1,z)$ 上

* 4：附着在坐标为 $(x,y-1,z)$ 上

* 5：附着在坐标为 $(x,y,z-1)$ 上

如果放置位置没有方块且附着位置上存在不为火把的方块或为地面，则合法。

---

##### 破坏

$DESTROY\ x\ y\ z$ 参数表示破坏位置

如果破坏位置上存在方块，则合法。

---

按照如下形式输出《世界》状态：

```
OUTPUT "THE WORLD\n"
FOR k FROM 0 TO Z-1
    FOR i FROM 0 TO X-1
        FOR j FROM 0 TO Y-1
            IF stone on (i,j,k) THEN
                OUTPUT 'r'
            ElSE IF sand on (i,j,k) THEN
                OUTPUT 's'
            ELSE IF torch on (i,j,k) THEN
                IF f IS 1 THEN
                    OUTPUT 'v'
                ELSE IF f IS 2 THEN
                    OUTPUT '^'
                ELSE IF f IS 3 THEN
                    OUTPUT '>'
                ELSE IF f IS 4 THEN
                    OUTPUT '<'
                ELSE
                    OUTPUT 't'
                ENDIF
            ELSE
                OUTPUT 'a'
            ENDIF
        ENDFOR
        OUTPUT '\n'
    ENDFOR
    OUTPUT '\n'
ENDFOR
```

### 输入格式

第一行三个整数 $X,Y,Z(1\leq X,Y,Z\leq10)$ 表示世界的大小。

第二行一个整数 $N(1\leq N\leq 10^5)$ 代表指令数量。

随后 $N$ 行，每行一条指令，具体格式见题目描述。

对于所有指令，**保证** $(x,y,z)$ 在《世界》内部。

### 输出格式

对于每条指令，输出一行 GOODJOB 或 AREYOUKIDDINGME。

最后按照格式输出《世界》状态。
