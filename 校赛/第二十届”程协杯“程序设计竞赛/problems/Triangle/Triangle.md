# 古希腊掌管三角形的神
- 每组测试用例的时间限制为 $5$ 秒
- 每组测试用例的内存限制为 $256$ 兆字节

## 题目描述
秧歌 (又被称为年轻的云) 是古希腊掌管三角形的神，无所畏惧的你来到秧歌的面前想要挑战他的权威，秧歌不屑一顾地对你说：“何意味？回答正确这个问题你才有资格触碰神圣的三角形”。

秧歌将 $n$ 根木棍摆在你的面前，编号从 $1$ 到 $n$，第 $i$ 根木棍的长度为 $a_i$。接着他会问你 $q$ 个对木棍的查询。每个查询会给出两个整数 $l$ 和 $r$ ( $1 ≤ l < r ≤ n$，且 $r - l + 1 ≥ 6$ )，请判断能否从编号为 $l$ 到 $r$ 的木棍中，挑选出 $6$ 根不同的木棍，组成 $2$ 个**非退化三角形**。

∗ 若一个三角形的三边长 $a、b、c$ 满足以下条件，则称其为**非退化三角形**：
$a < b + c、b < a + c$ 且 $c < a + b$。

## 输入格式
第一行包含两个整数 $n$ 和 $q$ ( $6 ≤ n ≤ 10^5，1 ≤ q ≤ 10^5$)，分别表示木棍的数量和查询的数量。
第二行包含 $n$ 个整数 $a_1, a_2, …, a_n$ ( $1 ≤ a_i ≤ 10^9$ )，其中 $a_i$ 表示第 $i$ 根木棍的长度。
接下来 $q$ 行，每行包含两个整数 $l$ 和 $r$ ( $1 ≤ l < r ≤ n$，且 $r - l + 1 ≥ 6$ )，表示每个查询的参数。

## 输出格式
对于每个查询，若能组成 $2$ 个非退化三角形，则输出 “YES” ( 不包含引号 )，否则输出 “NO” ( 不包含引号 )。

## 示例

**输入：**
```text
10 5
5 2 2 10 4 10 6 1 5 3
1 6
2 7
2 8
5 10
4 10
```
**输出：**
```text
YES
NO
YES
NO
YES
```

备注
- 第一个查询中，木棍长度为 $[5, 2, 2, 10, 4, 10]$，可选择两组木棍 $[2, 4, 5]$ 和 $[2, 10, 10]$，分别组成 $2$ 个非退化三角形。
- 第二个查询中，木棍长度为 $[2, 2, 10, 4, 10, 6]$，经验证无法组成 $2$ 个非退化三角形。
- 第三个查询中，木棍长度为 $[2, 2, 10, 4, 10, 6, 1]$，可选择两组木棍 $[1, 2, 2]$ 和 $[4, 10, 10]$，分别组成 $2$ 个非退化三角形。
- 第四个查询中，木棍长度为 $[4, 10, 6, 1, 5, 3]$，经验证无法组成 $2$ 个非退化三角形。
- 第五个查询中，木棍长度为 $[10, 4, 10, 6, 1, 5, 3]$，可选择两组木棍 $[1, 10, 10]$ 和 $[3, 4, 5]$，分别组成 $2$ 个非退化三角形。
